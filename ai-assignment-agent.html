About This Project

A conversational AI teaching assistant for CS students that explains programming assignments without giving away the solution.

  • Guides students by explaining concepts, not code
  • Uses Gemini Flash API for LLM backend
  • Focus on academic integrity and helpfulness

Features & Tech

  • Python-based core logic
  • Gemini Flash API integration
  • Two-layer prompting for safe, helpful responses
  • Error handling and basic testing
