# Personal Portfolio Website

This is a simple personal portfolio website for Deetya Yerradoddi, a student at the University of Washington. The site showcases selected projects, provides information about me, and offers ways to get in touch.

## Features
- Scrollable landing page with About, Projects, and Contact sections
- Dedicated detail pages for each project
- Clean, modern design (HTML & CSS only)

## Projects Highlighted
- Pin It Now
- ai-assignment-agent
- canvas-reminder-bot
- FloralCode
- Rebox
- self-sync

## How to Use
Open `index.html` in your browser to view the portfolio. Click on any project to see more details.
