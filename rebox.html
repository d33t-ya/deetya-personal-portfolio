About This Project

A lightweight returns platform that makes sending back online purchases as easy as ordering them. Connects customers with local drivers for pickup and drop-off.

  • Return request flow: customer to driver to drop-off
  • Confirmation codes for secure handoff
  • Built as a prototype for last-mile logistics

Features & Tech

  • Frontend: TypeScript, Tailwind CSS, Vite
  • Backend/DB: Supabase
  • Prototype, not yet deployed
