About This Project

A gamified wellness companion and dashboard for self-reflection, habit tracking, and personal growth. Built with Vite, React, and shadcn-ui.

  • Daily check-ins, quests, and habit tracking
  • Customizable avatar and motivational features
  • Modern UI with Tailwind CSS and shadcn-ui

Features & Tech

  • Frontend: React, TypeScript, Vite, shadcn-ui, Tailwind CSS
  • Gamification: quests, points, and rewards
  • Deployed via Lovable.dev
