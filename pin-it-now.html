About This Project

A live, full-stack AI event discovery platform that centralizes local family-friendly events by location. Built for parents and caregivers to find local events easily.

  • Aggregates events from multiple sources using AI-powered scraping
  • Admin dashboard for internal management
  • Deployed via Cloudflare and Base44

Features & Tech

  • Frontend: JavaScript, HTML/CSS
  • AI Pipeline: Base44 invokeLLM, OpenAI API
  • Analytics: Google Analytics
  • Deployment: Cloudflare, Base44
