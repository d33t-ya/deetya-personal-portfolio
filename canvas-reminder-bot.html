About This Project

A simple Discord bot that sends assignment reminders for a class server, connected to a Canvas calendar.

  • Fetches assignments from Canvas calendar
  • Sends daily and weekly reminders in Discord
  • Detects new assignments and notifies users

Features & Tech

  • Python, discord.py, ics, dotenv
  • Automated reminders and encouragements
  • Customizable for any class or calendar
